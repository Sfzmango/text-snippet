## <b><ins> Description: </ins></b>
        
This is a program constructed with Javascript and Webpack. The program is designed to be able to let users type in code snippets into the application. The program also features the ability to use it in offline environments.
<br><br>      

## <b><ins> Installation: </ins></b>
        
The app is deployed on Heroku so it is not needed to install anything. If you would like to run it locally, please download all the files included in the repository. Afterwards, please open your terminal into the repository folder and utilize Node.js's npm install feature `npm i` to install all the used packages. Once that is completed run the build script `npm run build`.

## <b><ins> Usage: </ins></b>
        
To start the program, please open the directory on terminal and type in `npm start`. You will then be able to go to the proper routes to see the app. You should also be able to install the app to be able to access it without an internet connection.
